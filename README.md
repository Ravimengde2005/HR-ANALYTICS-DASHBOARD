# 📊 HR Analytics Dashboard

## 📌 Project Overview
This project is an HR Analytics Dashboard created using Tableau.  
It helps analyze employee data and understand key reasons for employee attrition.

---

## 🎯 Objectives
- Analyze employee attrition trends  
- Identify factors affecting employee turnover  
- Improve employee retention using data insights  

---

## 🛠️ Tools Used
- Tableau  
- Microsoft Excel  

---

## 📂 Dataset
The dataset includes:
- Employee Age  
- Department  
- Job Role  
- Education Field  
- Job Satisfaction  
- Attrition  

---

## 📊 Dashboard Features
- Attrition Overview  
- Department-wise Attrition  
- Age Group Analysis  
- Job Satisfaction Rating  
- Education Field Analysis  
- Gender-wise Attrition  

---

## 📈 Key Insights
- Some departments have higher attrition  
- Job satisfaction impacts employee retention  
- Certain age groups show higher turnover  

---

## 📁 Project Files
- HR ANALYTICS DASHBOARD.twb (Tableau file)  
- HR Dashboard.xlsx (Dataset)  
- HR background.pptx.png (Background image)  

---

## 🚀 How to Use
1. Download the Tableau file  
2. Open in Tableau Desktop  
3. Connect dataset if needed  
4. Explore dashboard
